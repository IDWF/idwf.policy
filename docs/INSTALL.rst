idwf.policy Installation
------------------------

To install idwf.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``idwf.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        idwf.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
